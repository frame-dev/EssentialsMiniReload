[SIZE=5][B]Overview[/B][/SIZE]

EssentialsMiniReload is a modern essentials plugin for Spigot, Paper, and Purpur servers. It brings together the most common server tools in one lightweight plugin, so you do not need a large stack of separate plugins for everyday commands.

The plugin includes homes, warps, spawn management, economy commands, kits, moderation tools, private messages, mail, nicknames, backpacks, portable utility stations, recipe helpers, inventory utilities, world controls, and more.

It is designed for Minecraft [B]1.20.6 and higher[/B] and runs on Java 17.

Need support for older Minecraft versions? Use the original EssentialsMini plugin:
[URL]https://www.spigotmc.org/resources/essentialsmini.82775/[/URL]

[SIZE=5][B]Features[/B][/SIZE]

[LIST]
[*][B]Spawn System[/B] - Set and teleport to the server spawn.
[*][B]Homes[/B] - Set, delete, and teleport to homes.
[*][B]Home GUI[/B] - Open a GUI to manage and select homes.
[*][B]Warps[/B] - Create, delete, list, and teleport to warps.
[*][B]Teleport Requests[/B] - TPA, TPAHere, accept, deny, and teleport toggles.
[*][B]Back Command[/B] - Return to your last death or previous location.
[*][B]Economy[/B] - Balance, pay, balance top, eco admin commands, and bank features.
[*][B]Vault Support[/B] - Economy integration through Vault.
[*][B]Kits[/B] - Create and use permission-based kits.
[*][B]Inventory Tools[/B] - Invsee, ender chest, backpack, trash, repair, item commands, and portable utility stations.
[*][B]Portable Utility Stations[/B] - Open crafting, anvil, grindstone, smithing table, cartography table, loom, and stonecutter menus anywhere.
[*][B]Recipe Helpers[/B] - Show item recipes and retrieve ingredients from crafted items.
[*][B]Moderation[/B] - Mute, temp mute, ban, temp ban, unban, global mute, maintenance mode, and clear chat.
[*][B]Private Messaging[/B] - Message, reply, spy, and message toggle commands.
[*][B]Mail System[/B] - Send and manage player mail.
[*][B]Nicknames[/B] - Set player nicknames, apply skins through ProtocolLib, reset nicknames, and view nickname lists.
[*][B]Command Help GUI[/B] - Use [CODE]/emhelp[/CODE] to browse every EssentialsMiniReload command in a paged GUI with hover lore for description, usage, aliases, and permission.
[*][B]Player Utilities[/B] - Fly, god mode, heal, feed, speed, AFK, time played, online/offline player lists.
[*][B]World Controls[/B] - Day, night, sun, rain, thunder, player time, player weather, lightning, and firework commands.
[*][B]Geyser Support[/B] - Support for Bedrock players through Geyser where available.
[*][B]Custom Messages[/B] - Configure and customize plugin messages.
[*][B]Multiple Storage Options[/B] - YAML, SQLite, MySQL, and MongoDB support depending on the feature.
[/LIST]

[SIZE=5][B]Installation[/B][/SIZE]

[LIST=1]
[*]Download the plugin jar.
[*]Place the jar inside your server's [CODE]plugins[/CODE] folder.
[*]Start or restart your server.
[*]Edit the generated configuration files in [CODE]plugins/EssentialsMini/[/CODE].
[*]Restart the server again after configuring.
[/LIST]

[SIZE=5][B]Configuration[/B][/SIZE]

The plugin generates configuration and data files automatically after the first server start.

Common files include:

[LIST]
[*][CODE]config.yml[/CODE]
[*]Language/message files
[*][CODE]locations.yml[/CODE]
[*][CODE]kits.yml[/CODE]
[*][CODE]mail.yml[/CODE]
[*][CODE]tempMutes.yml[/CODE]
[*][CODE]nicks.yml[/CODE]
[/LIST]

You can customize messages, storage options, economy settings, moderation behavior, and many other plugin features.

Existing [CODE]config.yml[/CODE] files are updated with missing default options and bundled comments on startup or [CODE]/essentialsmini reload[/CODE] without overwriting configured values.

For [B]/nick <nickname> <skin>[/B] troubleshooting, enable [CODE]skinDebug: true[/CODE] in [CODE]config.yml[/CODE], reload or restart the server, then check the console for [CODE][SkinDebug][/CODE] lines.

Skin refresh supports Spigot, Paper, and Purpur by trying Paper/Purpur profile APIs first and falling back to ProtocolLib/NMS profile updates on Spigot-style servers.
